Countdown Advance Demo

Rules

  • Pause button will pause the countdown
  • Resume button will resume the countdown
  • 1 min button will set the countdown to 1 min
  • Reset will set the countdown value to 7 second
  • Extra time button will add extra 1 min including remaining time
  • Cut time button will reduce 10 second form the countdown

Include JS

Two JS file is required to run this countdown. One is the jQuery jQuery.min.js and another file is jquery.countdown.min.js. We need to include it into the footer.

Note:

Note that var current_time = new Date().getTime(); is a time object. We must pass a time object as an argument to the countdown function in order to work properly.

  • Variable: Time variable is javaScript default date object variable which gives timestamp value.
  • Format:Time format should be like this var format = '%H:%M:%S';
  • ChangeFormat:Format can be changed like this: format = '%-d day%!d ' + format;
  • Events:There are lots of events available in order to manipulate the counter.
